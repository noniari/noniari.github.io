Итак, для того, чтобы создать свой одностраничный веб сайт через github, тебе необходимо создать репозиторий, у него необчное название: username.github.io где username - это твой ник на гитхабе. ВАЖНО! Репозиторий должен быть публичным, иначе сайт не будет работать. После создания репозитория, тебе необходимо запустить кодовое пространство, чтобы это сделать, необходимо перейти во вкладку "open menu" слева сверху, затем выбрать "code spaces" и нажать на кнопку "create code space on main". После этого, тебе необходимо создать файл с названием index.html и добавить в него код, вот базовый шаблон для одностраничного сайта: 

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Заголовок страницы</title>
    <link rel="stylesheet" href="style.css"> <!-- Подключение стилей -->
</head>
<body>
    <h1>Привет, мир!</h1>
    
    <script src="script.js"></script> <!-- Подключение скриптов -->
</body>
</html>

В этом шаблоне есть базовая структура для одностраничного сайта, ты можешь изменить заголовок страницы, добавить свои стили и скрипты. Для того, чтобы увидеть изменения на сайте, тебе необходимо сохранить изменения и запушить их в репозиторий через терминал, для этого тебе необходимо ввести следующие команды: <!-- git add . --> - эта команда добавляет все изменения в индекс, git commit -m "здесь ты пишешь комментарий к коммиту, например added new changes ", после коммита мы должны запушить изменения в репозиторий, для этого необходимо ввести команду git push origin main . После того, как ты запушишь изменения, тебе необходимо подождать несколько минут, чтобы сайт обновился и отобразил твои изменения.

Готово! Теперь ты можешь открыть свой сайт по адресу https://username.github.io и увидеть свои изменения. Не забывай, что ты можешь добавлять больше информации о себе, своих интересах и проектах, а также использовать различные стили и скрипты для улучшения внешнего вида и функциональности сайта.
